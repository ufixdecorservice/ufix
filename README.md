# Ufix Decor Service Website

A production-ready single-page website for **Ufix Decor Service Co., Ltd.**

## Tech Stack
- HTML5 / CSS3 / Bootstrap 5
- Vanilla JavaScript
- LocalStorage CMS
- PWA Support
- Multilingual (TH/EN)

## Hosting on GitHub Pages
1. Push all files to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select the branch and folder (`root`) to deploy.

## Admin CMS
- URL: `admin.html`
- Default Username: `admin`
- Default Password: `admin123`
- *Note: CMS data is stored in your browser's LocalStorage.*
